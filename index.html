Markdown記法

2014-12-25 01:36

よく使う記法のみ抜粋。

文字装飾

見出し

文頭に「#」で見出し。

  # h1
  ## h2
  ### h3
  #### h4
  ##### h5
  ###### h6

強調

「__」か「**」で囲むと太字。

「_」か「*」で囲むと斜体。

**太字**
__太字__
*斜体*
_斜体_

打ち消し線

「~~」で囲むと打ち消し線。

~~打ち消し線~~

リスト

Disc型

「-」「+」「*」のいずれかでDisc型リスト。

- リスト
+ リスト
* リスト

Decimal型

「数値.」でDecimal型リスト。

1. リスト
2. リスト

コード

半角スペース4つでコードブロック。
あるいは段落を「```」で囲む。
文中に記述するならば「`」でその箇所を囲む。

```
コードブロック
```

文中に`コード`を含める場合

引用

文頭に「>」で引用。

> 引用文

リンク

タイトルあり

[リンクテキスト](URL "タイトル")

タイトル無し

[リンクテキスト](URL)

水平線

以下のパターンで水平線が表示される。
（他にもパターンあり）

***
* * *
- - -

テーブル

以下のパターンでテーブル表示。

| left | center | right |
|:-----|--------|-------|
| 1    |   2    |     3 |
| a    |   b    |     c |

その他

  • 文末に半角スペース2つで段落内改行。
  • バックスラッシュ「\」を記号の前に記述すると、markdown記法を無効化できる。

CartonによるPerlモジュールの管理方法

2014-12-23 01:33

とりあえず基本的な部分だけ。

cpanfile作成

% cat > cpanfile  
requires '<Perlモジュール名>';

requires '<Perlモジュール名>', '<バージョン>';と書くことでバージョンの指定も可能。

Perlモジュールインストール

cpanfile.snapshotが追加される。

% cpan install

--deploymentフラグを付けると、利用可能なモジュールとバージョンのみをインストールする。

PerlモジュールをGit管理下より外す

local/ 以下にCarton管理下のモジュールが設置される為、これは管理から外す。

% echo local/ >> .gitignore

Perlモジュールを使用する

% carton exec <Perlモジュール名>

エイリアスでcarton execをce等にすると楽。
